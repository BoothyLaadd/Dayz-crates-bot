Quick start (Replit or Local)

On Replit:
1) Import this project (via ZIP or GitHub).
2) Add secrets in Replit (padlock icon):
   - BOT_TOKEN
   - CLIENT_ID
   - (optional) GUILD_ID
   - (optional) OPEN_CRATE_CHANNEL_ID
3) Open the Shell and run: npm install
4) Click Run. Copy the web URL and set up an UptimeRobot HTTP monitor every 5 minutes to keep it awake.

Locally:
1) Extract files.
2) Copy .env.example to .env and fill values.
3) Run: npm install
4) Start bot: npm start
